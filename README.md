# Damp

A falling-sand / pixel-simulation sandbox. Build sandcastles by mixing sand and water — wet sand sticks, dry sand flows.

Built with Python + [pygaminal](https://github.com/enesbaytekin/gaminal) on top of pygame.

## Controls

| Key | Action |
|-----|--------|
| `1` | Dry sand |
| `2` | Water |
| `3` | Pre-wetted sand |
| `4` | Water eraser |
| Left click | Paint |
| Right click | Erase |
| `Ctrl` + drag | Draw/erase a line (both buttons) |
| `-` / `+` / scroll | Brush radius |
| `F11` | Toggle fullscreen |

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
